Ejemplo de estilo interno (Css en el <style>)

En este ejemplo se muestra como aplicar estilos mediante Css interno, es decir, escribiendo las reglas de estilo dentro de la etiqueta <style>. en la seccion <head> del documento Html . Esta forma permite centralizar los estilos de una pagina y mantmantener separados el contenido (Html) de la presentación (Css), fasilitando el mantenimiento y la organizacion.
